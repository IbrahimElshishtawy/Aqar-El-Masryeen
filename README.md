## Overview

**El Masryeen** is a modern real estate management platform designed to simplify the management of property units, customer payments, installment plans, and financial tracking.

The project helps real estate businesses and property managers organize daily operations in a clear, structured, and scalable way. It reduces manual work, minimizes financial errors, and provides a better experience for both administrators and users.

---

## Vision

The goal of **El Masryeen** is to transform traditional property management into a smarter digital workflow by providing:

- Clear unit management
- Accurate financial tracking
- Flexible installment handling
- Organized payment records
- Better visibility for projects and partnerships

---

## Core Features

### Unit Management
- Manage all real estate units in one place
- Display full unit details:
  - Unit type
  - Floor
  - Area
  - Payment system
  - Availability status

### Financial Summary
- Final contract value
- Down payment
- Total paid amount
- Remaining amount
- Unit financial status

### Installment Management
- Generate installment sheets based on the number of installments
- Track each installment separately
- Display:
  - Installment number
  - Due date
  - Installment value
  - Paid amount
  - Remaining amount
  - Installment status

### Payment Records
- Add new payments easily
- Store payment history with full details
- Link payments directly to installments
- Automatically update totals and balances

### Partner & User Tracking
- Support linked partner accounts
- Show expenses and records for both user and partner
- Improve transparency in shared operations

### Real-Time Updates
- Instant data refresh
- Updated financial calculations
- Better operational visibility

## Problem Solved

Traditional real estate tracking often suffers from:

- Manual calculations
- Missing payment records
- Difficulty tracking installment status
- Poor visibility over unit performance
- Confusing financial summaries

**El Masryeen** solves these issues by offering a clean, digital, and organized management experience.

---

## Workflow

1. Create or register a new unit
2. Define the payment method
3. Enter down payment details
4. Set the number of installments
5. Generate installment sheet
6. Record payments manually by the user
7. Update installment and payment status automatically
8. Track the full financial state of the unit
